HTML

  • HTML stands for Hyper Text Markup Language.
  • It is responsible for the structure of a web page.
  • HTML is made up of elements, which is a type of content on a webpage.
  • Commonly used elements include: text, image, header, and list elements.
  • The head element contains important information (metadata) about the web page that help the browser render correctly. Elements included are not visible to a user on the webpage.
  • The body element contains the visible content shown to the user.
  • Elements are also used to divide content on a page, and to describe content to browsers and other developers.
  • Common dividing/describing elements include: meta-related elements, content division elements, and semantic elements.
  • The declaration < !DOCTYPE html > tells the browser to expect HTML on the page.

CSS

  • CSS stands for Cascading Style Sheets.
  • It is responsible for the appearance of a web page.
  • A CSS document should be linked to an HTML document within the head element using a style tag.
  • CSS can be applied as an internal stylesheet, external stylesheet, or inline.
  • Inline CSS can be helpful when a developer wants to style an element directly in the HTML file, using a style attribute. They are not as flexible as external/internal style sheets, because each inline style must be edited individually for each change.
  • Internal stylesheets are located in the head section of the HTML file. The rules will only apply to that page.
  • External stylesheets are created in an external CSS file and are the most preferred method.
  • External stylesheets are linked to the file using a link element placed in the HTML head section.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Classes and IDs are used to identify elements.
  • A CSS rule is composed of a selector, which defines the element or attribute the rule will apply to, and a declaration block, which will apply a property and value.
  • Using * as a selector will apply a rule to all visible elements.

Git

  • Git is an open source version control system.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is an object-oriented programming (OOP) language.
  • JavaScript is used for incorporating logic into a webpage for user interaction.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is typically executed from top to bottom, but can be interrupted using conditional statements such as if statements.
  • A string variable is a sequence of text and requires quotations.
  • Number variables and Boolean variables (true/false) do not require quotations.
  • An array is a variable that allows you to store multiple values in one reference.
  • JavaScript is case sensitive.
  • Single line comments are written with two slashes //
  • Multiple line comments are written between /* and */ .
  • A for loop allows you to repeat code without having to write it several times.
  • Conditionals are used to test if an expression is truthy or falsy.
  • Functions are used to package functionality that can be reused by calling in the code.
  • console.log() is used to output a message to the web console by adding an argument between parentheses.
  • x++ is a shortcut for x+1.
