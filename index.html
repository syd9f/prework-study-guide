Prework Study Guide

HTML

  • HTML stands for Hyper Text Markup Language.
  • It is responsible for the structure of a web page.
  • The head element contains information about the web page.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stands for Cascading Style Sheets.
  • It is responsible for the appearance of a web page.
  • A CSS document should be linked to an HTML document within the head element using a style tag.

Git

  • Git is an open source version control system.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is used for incorporating logic into a webpage for user interaction.
